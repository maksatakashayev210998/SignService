## libs

Сюда положите локальные JAR'ы криптопровайдера, которые не подтягиваются из Maven Central.

Минимально для сборки этого сервиса нужны классы провайдера `kz.gamma.*` (например `GammaTechProvider.jar`).

Файлы `*.jar` в этой папке **игнорируются** в `.gitignore`, чтобы не хранить крипто-библиотеки в репозитории.
